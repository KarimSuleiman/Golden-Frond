## About This Project

Golden Frond (السعفة الذهبية) is a full-stack web app I built for a real car 
import/trading business in Jordan. Customers can track purchased vehicles 
through the import and shipping process (status, container number, booking 
number, tracking link), browse a car marketplace with detailed filtering, 
and admins manage incoming shipments, listings, and user roles through a 
dedicated dashboard. The entire UI is in Arabic with RTL support.

Stack: React + TypeScript on the frontend, Express/Node on the backend, 
PostgreSQL with Drizzle ORM, role-based access control (user/trader/admin), 
session-based auth with bcrypt password hashing, file uploads, and 
transactional email.

This project is where I leveled up at treating AI as a real development 
partner rather than a black box — writing precise, structured prompts to 
get the implementation I actually wanted, then reading, testing, and 
debugging the generated code myself instead of accepting it blindly.

Skills gained:
- Full-stack architecture (React + Express + PostgreSQL)
- Relational database design and ORM usage (Drizzle)
- Role-based access control and session-based authentication
- RESTful API design with type-safe, schema-validated contracts (Zod)
- File upload handling and transactional email integration
- Reading and debugging AI-generated code rather than copy-pasting it
- Iterative, spec-driven prompting to direct an AI coding assistant
- Deploying and operating a live full-stack app on Replit
